# Campus-assistant-using-gen-ai
An AI-based campus assistant that answers common VIT Vellore student queries using document retrieval and a pre-trained language model.
## Features

- Answers campus-related questions (hostel, library, attendance, exams)
- Retrieves relevant information before generating responses
- Supports interactive chat-based queries
- Handles follow-up questions using conversational memory

---

## Tools & Technologies

- Python  
- Google Colab  
- LangChain  
- ChromaDB  
- HuggingFace embeddings  
- Google-hosted open LLM (Gemma)

---

## How to Run

1. Open the Google Colab notebook linked below  
2. Run the cells in order  
3. Start the chat loop and ask campus-related questions
